== Description ==

Fashion Boutique is a phenomenal, multipurpose and modern theme that is suitable to create website for fashion boutique, vintage boutique, clothing store, toy store, lingerie, accessories, brands, urban clothing, fabrics, jewellery store, etc. You can showcase your work in style and grab the attention of a wider audience using this gorgeous theme. The theme adjusts well and looks stunning on all devices screen sizes. With the social media option you can present all your social pages and give credibility to your business. To boost your siteâ€™s functionality, you are allowed to add custom codes like CSS/JS and available shortcodes to include custom post types. 100% clean and secure codes are used in developing this theme. Implementation on bootstrap framework makes it user-friendly and strong. Visitors can translate pages into several languages available to read content in their preferred language. With SEO optimization your site is ready to rank on all search engines. Moreover, it is optimized for speed and cross-browser compatible to load on all browsers. The theme comes with detailed documentation of its features and guidelines on installation. You can choose from pre-built layouts available or create a custom layout for pages. You can reorder sections and also enable/disable all sections.
